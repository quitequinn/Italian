# Mexican handpainted signage — reference

186 photographs of hand-painted shop signage shot in the Yucatán, Mexico
(March 2012). This is the primary visual source for the Italian typeface — the
bold, reverse-contrast slab letterforms common to this signage are the
"Italian" / French Clarendon genre the design draws from.

## Contents

- `contact-sheet.html` — browsable grid of every photo (open in a browser).
- `thumbs/` — 600px web-sized thumbnails (committed, ~13 MB).

## Full-resolution originals

The ~5 MB/photo originals are **not** committed (they would bloat the repo).
They live in Google Drive:

`My Drive/QT Clients/Quite Type/2012 Mexican handpainted signage/`

Trace letterforms from the full-res originals; use the thumbnails only to
browse and choose. Thumbnail filenames mirror the originals (subfolders are
flattened with `__`, e.g. `03.23.12__P1020xxx.jpg`).

## Using this material

These photos contain many different sign-painters' hands and styles. A coherent
typeface comes from committing to one style direction, not averaging all of
them — expect 2-3 distinct designs to come out of this set. See the repo README
for the curate → vectorize → build pipeline.
